Actaria

Actaria is a lightweight IDE and game engine for creating visual novels. 
It allows you to design scenes, dialogues, branching storylines, triggers, 
and manage assets (backgrounds, characters, sounds) through an intuitive GUI. 
Once your story is ready, you can export it as a standalone executable.

Features:
Scene Editor: create and manage multiple scenes with descriptions and backgrounds.
Dialog Tree Editor: build branching dialogues with conditional paths and triggers.
Event System: add game logic to scenes and dialogues.
Asset Management: assign images, music, and sound effects to scenes, characters, and dialogues.
Serialization: save and load your projects easily.
Export: compile your visual novel into a standalone executable.

Technology:
C++ with Qt for GUI development.
Object-oriented architecture for entities such as scenes, dialogues, characters, and triggers.
Custom serialization methods for saving and loading game data.

Getting Started:
1. Clone the repository:
   git clone https://github.com/DanilaEfimov/Actaria.git
2. Open the project in Qt Creator.
3. Build and run the IDE.
4. Start creating visual novels by adding scenes, dialogues, and assets.
